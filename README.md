# Fraud Detection with Machine Learning

Este proyecto tiene como objetivo detectar transacciones fraudulentas utilizando modelos de Machine Learning.

## 📊 Contenido del proyecto

- Análisis Exploratorio de Datos (EDA)
- Preprocesamiento de datos
- Modelos base (Logistic Regression, Random Forest, XGBoost)
- Optimización de modelos
- Interpretabilidad con SHAP

## 📁 Estructura del repositorio

- `notebooks/`: notebooks del proyecto
- `images/`: gráficas y visualizaciones
- `docs/`: documento final
- `data/`: instrucciones del dataset

## ⚙️ Tecnologías utilizadas

- Python
- Scikit-learn
- XGBoost
- SHAP
- Pandas / NumPy

## 📈 Resultados

El modelo Random Forest obtuvo un **F1-score de 0.83**, mostrando un buen desempeño en la detección de fraude.

## ⚠️ Nota

La hipótesis inicial (F1 = 0.85) no se cumplió, lo cual fue reportado de manera transparente como parte de la integridad académica del estudio.
